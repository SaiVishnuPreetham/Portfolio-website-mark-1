I Created.

Face Detection in real time using OpenCV

I created a face detection model using OpenCV and Python. This project implements a real-time face detection system using Python and OpenCV. It uses Haar cascade classifiers to identify and track faces in live video streams efficiently. The system can detect multiple faces simultaneously and works with varying lighting and angles. It serves as a foundational project for exploring advanced computer vision tasks like face recognition. Ideal for security, authentication, and surveillance applications.

In addition to detection, the model also counts the number of faces in real-time. This feature enhances the system's capability to provide instant analytics, making it suitable for applications requiring crowd monitoring or real-time statistics.
